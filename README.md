# Classificação Binária de Textos com LIME e SHAP

## Descrição
Projeto acadêmico desenvolvido na disciplina Tópicos Especiais em Programação (TEP) do curso de Análise e Desenvolvimento de Sistemas (ADS). O trabalho aborda o processo completo de classificação binária de textos, com foco em interpretabilidade e explicabilidade em Inteligência Artificial.

## Objetivo
Construir um pipeline de classificação de textos utilizando técnicas de Processamento de Linguagem Natural e aplicar métodos de explicabilidade para compreender como o modelo toma decisões.

## Pipeline 
1. Pré-processamento dos textos  
2. Vetorização com Bag-of-Words e TF-IDF  
3. Treinamento de modelos lineares  
4. Avaliação do desempenho  
5. Explicação das predições com LIME e SHAP  

## Interpretabilidade
### LIME
Utilizado para gerar explicações locais das predições, analisando no mínimo três exemplos distintos.

### SHAP
Aplicado para gerar explicações locais e globais, incluindo o summary plot para análise global do modelo.
